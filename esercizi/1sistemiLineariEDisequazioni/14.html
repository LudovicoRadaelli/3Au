Esercizio 1

Svolgere il seguente sistema usando il metodo di sostituzione \[ \begin{cases} 2x + 5y = 7 \\\\ -3x + 4y = 1 \end{cases} \]
Soluzione: \[ \begin{cases} x = 1 \\\\ y = 1 \end{cases} \]




Esercizio 2

Svolgere il seguente sistema usando il metodo di riduzione \[ \begin{cases} 3x + 5x = 1 \\\\ -2x + 2y = 3 \end{cases} \]
Soluzione: \[ \begin{cases} x = \dfrac{1}{8} \\\\ y = \dfrac{13}{8} \end{cases} \]




Esercizio 3

Svolgere il seguente sistema usando un metodo a scelta \[ \begin{cases} \left(2x + 1\right)^2 -5\left(1 -y\right) = 4x^2 -y \\\\ 5x + \left(2x + 1\right)\left(y - 2\right) = 2xy + 1 \end{cases} \]
Soluzione: \[ \begin{cases} x = 7 \\\\ y = -4 \end{cases} \]
