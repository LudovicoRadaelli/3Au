Compiti per casa


Esercizio 1


Dopo averlo portato in forma normale, risolvere il seguente sistema con il metodo di sostituzione \[ \begin{cases} x - 8y = 6 \\\\ y - 3 = x - y \end{cases} \]
Soluzione
\begin{cases} x = -6 \\\\ y = -\dfrac{3}{2} \end{cases}




Esercizio 2


Dopo averlo portato in forma normale, risolvere il seguente sistema con il metodo di riduzione. \[ \begin{cases} \dfrac{2x - y}{3} - 2 = x \\\\ \dfrac{1}{6}x + \dfrac{y - 1}{3} = -\dfrac{3}{2} \end{cases} \]
Soluzione
\begin{cases} x = -5 \\\\ y = -1 \end{cases}




Esercizio 3


Dopo averlo portato in forma normale, risolvere il seguente sistema utilizzando un metodo a piacere. \[ \begin{cases} \dfrac{x + 4}{3} +\dfrac{3}{2}(x + y) = 0 \\\\ \left(\dfrac{1}{2}x - 1\right)\left(1 + 2x\right) + y = x^2 + 2 \end{cases} \]
Soluzione
\begin{cases} x = -\dfrac{10}{7} \\\\ y = \dfrac{6}{7} \end{cases}
