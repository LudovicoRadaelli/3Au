Consideriamo la retta associata all'equazione \[ y = (-3a + 2)x - 4a \] con \(a\) numero qualsiasi.

  • Stabilire per quale valore di \(a\) la retta è orizzontale.

  • Stabilire per quale valore di \(a\) la retta è crescente.

  • Stabilire per quale valore di \(a\) la retta è parallela alla retta di equazione \[ 2x + 5y -1 = 0 \]

Esercizio 1


Consideriamo la retta associata all'equazione \[ y = (-3a + 5)x + 4a + 1 \] con \(a\) numero qualsiasi.

  • Stabilire per quale valore di \(a\) la retta è orizzontale.

    Soluzione:\(\,\,a = \dfrac{5}{3}\)

  • Stabilire per quale valore di \(a\) la retta è decrescente.

    Soluzione:\(\,\,a \geq \dfrac{5}{3}\)

  • Stabilire per quale valore di \(a\) la retta è parallela alla retta di equazione \[ -6x + 3y -1 = 0 \]
    Soluzione:\(\,\,a = 1\)
